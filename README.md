# Orklas Frontend

Orklas is a Room reservation application, This repository is an implementation of its frontend. Using react-ts, it allows giga fast development speed with no compromise on its edge cutting technology.

## Development

To develop on a newer orklas api spec, please generate a new `typescript-axios` client code and paste the output into src/api directory.

## Set up and run

First, install the dependencies

`npm install`

then run it

`npm run dev`
